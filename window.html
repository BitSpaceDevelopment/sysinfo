About This Application

This application allows you to monitor the system information for your device. You will have access to:

  • System Memory
  • CPU Usage
  • System Storage (coming soon)
  • Display information

This allows you to have a standard system information tool across all operating systems where Chrome is supported.

App developed by Dan Blair.

Check out the app homepage sysinfoapp.com
